# Tunapp Blog Cover Generator

Generate iOS-emoji blog covers for Tunapp — 740 × 480 px.

**Live:** https://Normalien24.github.io/tunapp-cover-generator/

## Cara pakai

1. Buka link di atas
2. Paste judul blog (satu per baris), **emoji di akhir judul**
3. Pilih warna awal jika perlu
4. Klik **Generate Covers**
5. Download tiap cover atau semua sekaligus

**Format judul:**
```
146. Armenian and the Diaspora: Language as Identity and Survival 🌍🧬
147. Learning Armenian at Home: Tips for Busy Families 📚✨
148. Music in Armenian Culture: Songs That Survived Centuries 🏛️🎶
```
